# 开发环境

## conda建立虚拟环境

```bash
conda create -n py37 python=3.7
```

## 激活虚拟环境

```bash
conda activate py37
```

## 安装使用的Python包

```bash
pip install SciPy numpy==1.19.3 PySide2 schemdraw scikit-rf
```
